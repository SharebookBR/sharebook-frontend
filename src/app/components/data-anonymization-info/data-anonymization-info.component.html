Anonimização de Dados

1. O que é Anonimização?

A anonimização é o processo de exclusão definitiva e irreversível de dados pessoais de sua conta no Sharebook. Este procedimento remove permanentemente informações como nome, e-mail, foto de perfil e outros dados pessoais, transformando suas interações em completamente anônimas no sistema.

2. Características do Processo

Processo Imediato e Irreversível:

  • A anonimização é executada imediatamente após a confirmação
  • Não é possível reverter o processo após sua conclusão
  • Desconexão automática do sistema após confirmação
  • Necessidade de criar uma nova conta caso deseje retornar ao Sharebook

3. Dados Afetados pela Anonimização

Os seguintes dados serão permanentemente removidos:

  • Dados pessoais: Nome completo, e-mail, telefone
  • Perfil: Foto de perfil, biografia, preferências
  • Identificadores: Todos os identificadores que permitam sua identificação

Dados que permanecem anônimos:

  • Reviews e avaliações (sem identificação do autor)
  • Comentários e interações (de forma anônima)
  • Dados estatísticos agregados (não identificáveis)

4. Fundamento Legal

O direito à anonimização está fundamentado na Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018), especificamente no artigo 18, que assegura ao titular dos dados pessoais o direito de obter do controlador:

  • Eliminação dos dados pessoais tratados com consentimento do titular
  • Anonimização de dados desnecessários, excessivos ou tratados em desconformidade com a LGPD
  • Revogação do consentimento para tratamento dos dados

5. Seus Direitos

Como titular dos dados, você tem:

  • Autonomia total: Liberdade completa sobre sua conta e dados pessoais
  • Decisão informada: Direito de ser informado sobre as consequências da anonimização
  • Suporte: Assistência através de nossos canais de atendimento
  • Transparência: Informações claras sobre o processo e seus efeitos

6. Antes de Anonimizar sua Conta

⚠️ Importante - Leia Atentamente

A anonimização é uma ação definitiva e irreversível. Antes de prosseguir:

  • Certifique-se de que realmente deseja remover todos os seus dados pessoais
  • Faça backup de informações importantes (como lista de livros)
  • Considere alternativas como desativação temporária da conta
  • Entre em contato conosco se tiver dúvidas sobre o processo

7. Como Executar a Anonimização

Para anonimizar sua conta no Sharebook:

  1. Acesse sua conta no sistema
  2. Navegue até as configurações da conta
  3. Procure pela opção "Anonimizar Conta" ou "LGPD"
  4. Leia cuidadosamente os termos e consequências
  5. Confirme com sua senha atual
  6. Forneça uma razão para a anonimização (mínimo 30 caracteres)
  7. Confirme a ação - irreversível

8. Após a Anonimização

Após confirmar a anonimização:

  • Você será automaticamente desconectado do sistema
  • Todos os seus dados pessoais serão imediatamente removidos
  • Suas contribuições permanecerão de forma anônima
  • Não será possível recuperar a conta ou os dados removidos
